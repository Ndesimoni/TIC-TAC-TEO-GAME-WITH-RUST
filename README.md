## üïπÔ∏è How to Play
#### Player Selection

Press Tab to move left and right between X and O

Press Enter to confirm your selection

Press Esc to quit the game at any time

Gameplay

Use Tab to navigate between board squares

Press Enter to place your mark

The computer automatically selects its move after yours

The game ends when there is a winner or a draw

Quit Game

Press Esc at any point to exit the game


| Key     | Action                         |
| ------- | ------------------------------ |
